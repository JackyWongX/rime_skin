# Rime 小狼毫皮肤在线设计器设计

## 目标

在当前目录实现一个纯 HTML 单文件应用，用于设计小狼毫 `weasel.custom.yaml` 皮肤。用户可以粘贴或导入现有预设代码，修改皮肤颜色，并实时预览小狼毫候选窗效果，最后复制或下载更新后的 YAML。

## 范围

第一版只支持小狼毫配置，重点处理 `weasel.custom.yaml` 中的 `style/color_scheme` 与 `preset_color_schemes` 颜色字段。页面不依赖构建工具、后端服务或外部库，最终交付一个可直接用浏览器打开的 `index.html`。

不处理鼠须管配置、复杂 YAML anchor、条件合并、跨平台皮肤差异或完整 YAML 语义校验。

## 页面结构

采用三列工作台。

左侧是 YAML 代码区，包含文件导入按钮、粘贴/编辑用的文本区域、解析按钮、加载示例按钮。用户导入 `.yaml`、`.yml` 或 `.txt` 文件后，文件内容显示在文本区域中。

中间是颜色控制区。应用从 YAML 中识别颜色字段并生成编辑项。每个编辑项包含字段名、中文说明、颜色选择器、HEX 输入、RGB 输入。用户修改任一输入后，其它输入、YAML 文本和右侧预览同步更新。

右侧是实时预览区。预览模拟 Windows 桌面中的小狼毫候选窗，展示编码文本、候选文字、高亮候选、注释文字、背景、边框和阴影。右侧还提供复制 YAML、下载 YAML 和重置示例入口。

## 颜色字段

内置识别常见小狼毫颜色字段：

- `back_color`: 候选窗背景
- `border_color`: 候选窗边框
- `text_color`: 编码文本
- `candidate_text_color`: 候选文字
- `hilited_candidate_text_color`: 高亮候选文字
- `hilited_candidate_back_color`: 高亮候选背景
- `hilited_text_color`: 高亮编码文本
- `hilited_back_color`: 高亮编码背景
- `comment_text_color`: 注释文字
- `hilited_comment_text_color`: 高亮注释文字
- `label_color`: 序号文字
- `hilited_label_color`: 高亮序号文字

如果 YAML 中出现其它以 `_color` 结尾的字段，也显示为可编辑项，说明使用“自定义颜色字段”。

## 解析与生成

第一版使用轻量文本解析，不引入完整 YAML 解析器。解析器按行扫描，识别 `key: value` 形式中 key 以 `_color` 结尾的字段。

支持的颜色输入格式：

- `0xRRGGBB`
- `#RRGGBB`
- `rgb(r, g, b)`
- `r,g,b`

内部统一转换为 `{ r, g, b }` 与 `#RRGGBB`。写回 YAML 时默认使用小狼毫兼容的 `0xRRGGBB`。当修改已识别字段时，只替换对应行的颜色值，保留原始缩进、字段顺序和其它配置内容。

如果用户手动新增颜色字段，应用追加到当前选中的配色方案块中。若无法可靠定位配色方案块，则提示用户先粘贴包含 `preset_color_schemes` 的配置，或使用内置示例开始。

## 交互流程

首次打开页面时加载内置示例配置，并展示对应颜色项和预览。

用户粘贴或导入 YAML 后点击解析，应用扫描颜色字段。解析成功后刷新颜色控制区和预览。未识别到颜色字段时，保留文本内容并显示提示。

用户通过颜色选择器、HEX 输入或 RGB 输入修改颜色。输入合法时即时同步；输入非法时仅标记当前输入，不写回 YAML，也不更新预览。

用户点击复制 YAML 时，将当前文本区域内容复制到剪贴板。用户点击下载 YAML 时，生成 `weasel.custom.yaml` 文件。

## 错误处理

导入文件读取失败时显示错误提示，不清空当前内容。

颜色格式非法时在当前字段附近显示错误状态，并保留上一次有效颜色。

解析不到颜色字段时给出明确提示，并提供加载示例配置的入口。

浏览器不支持剪贴板 API 时，提示用户手动复制左侧文本区域内容。

## 视觉设计

整体界面偏工具型，强调密度和可扫描性。三列布局在桌面端并排显示；窄屏下改为单列堆叠，顺序为代码、颜色、预览。

色彩使用中性浅色背景、清晰边框和高对比文本。预览区使用深色桌面背景突出候选窗。控件尺寸固定，避免颜色值变化造成布局跳动。

## 验证方式

实现后直接用浏览器打开 `index.html` 验证。检查导入、粘贴解析、颜色修改、YAML 写回、复制、下载和响应式布局。

根据用户要求，不创建或执行测试代码文件。
